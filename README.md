# Twisted-IRC-server
This is basic IRC Server implementation based on Twisted python library.

Procedure to Setup Server:
- Install Python (Duh)
- Install pip
- Install twisted (pip install twisted)
- Install Service Indetity (pip install service_identity)

To run the server:
python ircd.py
